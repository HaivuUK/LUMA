Thank you for wanting to help make LUMA more accessible!

Found an error in a translations? Then please report it by [opening an issue here](https://github.com/HaivuUK/luma/issues). (We do not have any yet.) 

Or, if you want to help translate LUMA documentation into another language, please follow the steps below.

### How To Help

#### 1. Fork the Repository and Create a New Branch
Fork the LUMA repository on GitHub to your own account.

Create a new branch for your translation work. For example, if you are translating to Spanish, you might name your branch `translate-es`.

#### 3. Translate the Documentation
Translate the documentation files in the `docs/content` + your ISO language code (e.g., `/es` directory) into your target language. 
Make sure to follow the existing structure and formatting of the files.

#### 4. Commit and Push Your Changes
Commit and push your changes with a clear and descriptive commit message. 
For example, "Translated documentation to Spanish."

#### 6. Create a Pull Request
Go to the original LUMA repository and create a pull request from your forked repository. 
Provide a clear description of the changes you made and the language you translated to. 
We will review your pull request and provide feedback if necessary. 
Once approved, your translation will be merged into the main repository and appear on the LUMA documentation site.
